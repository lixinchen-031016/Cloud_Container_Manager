# 更新日志

## [1.0.3] - 2026-03-19

### 新增
- **密码加密存储**: 支持保存 SSH 密码，使用 AES-256 加密 (#8)
- **SSH 终端模拟器**: 内置终端界面，支持执行 Linux 命令 (#9)
- **自动登录**: 已保存密码的服务器可自动连接 (#10)

### 改进
- 配置文件格式优化，支持 encrypted_password 和 encrypted_passphrase 字段
- 连接对话框增加"记住密码"复选框
- 保存的服务器点击后优先尝试自动连接

### 依赖
- 新增 cryptography>=41.0.0

## [1.0.2] - 2026-03-19

### 新增
- **服务器配置保存**: 自动保存服务器配置，下次运行时无需重新添加 (#5)
- **右键菜单**: 支持右键删除服务器配置 (#6)
- **调试输出**: 添加详细的调试信息，便于排查监控问题 (#7)

### 改进
- 改进了 SSH 连接流程，保存的配置会在下次连接时预填充信息
- 增强了监控数据的错误处理和日志输出
- 配置文件存储在 `~/.cloud_container_manager/servers.json`，权限设置为 600

## [1.0.1] - 2026-03-19

### 修复
- **修复 Python 3.11 兼容性**: 修复 `ssh_client.py` 中 f-string 包含反斜杠的语法错误 (#1)
- **修复 PyQt6 兼容性**: 将 `QMenu.exec_()` 改为 `QMenu.exec()` (#2)
- **修复字体警告**: 移除对特定字体的硬编码，使用系统默认字体避免警告 (#3)
- **增强错误处理**: 为所有系统信息解析器添加类型检查和空值处理，避免 `'int' object has no attribute 'strip'` 错误 (#4)

### 改进
- 系统解析器现在能正确处理 SSH 命令执行失败的情况
- 优化了 CPU、内存、磁盘、网络和系统负载的解析逻辑
- 提高了在不同 Linux 发行版上的兼容性

## [1.0.0] - 2026-03-19

### 新增
- 初始版本发布
- SSH 连接管理（支持密码和密钥认证）
- 服务器实时监控（CPU、内存、磁盘、网络、系统负载）
- Docker 容器管理（查看、启动、停止、重启、删除、查看日志）
- macOS 原生 GUI 界面
- 虚拟环境支持（Python 3.11）
- 应用打包脚本
